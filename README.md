# Core
Core project which adds native support for cordova project.

The project provides the following functionality
* Basic Widgets and Android Layouts
* Validation support
* Unidirectional Databinding framework
* Cross platform JSON
* Cross platform file system access
* Converters

Ashera support the following platforms:

OS            | Framework
------------- | -------------
Android       | Android
IOS           | J2objc
Windows       | SWT
Mac           | SWT
Linux         | SWT


The following widgets are supported by this plugin:

## Layouts
* [View]([http://www.google.fr/](https://developer.android.com/reference/android/view/View) "View")
* FrameLayout
* LinearLayout
* RelativeLayout
* ListView
* ListView
* HorizontalScrollView
* ScrollView
* TableLayout
* TableRow

## Widgets
* Button
* CheckBox
* Chronometer
* EditText
* ImageButton
* ImageView
* ProgressBar
* RadioButton
* RadioGroup
* Spinner
* Switch
* TextView
* ToggleButton
